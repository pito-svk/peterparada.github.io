Useful Vim commands

May 1, 2021

In this blog I am going to show you commands that I've found useful and some new that might help you become better at using Vim.

I am writing this mostly for myself so I can reference here but also learn some new commands I haven't used yet.

Go to the end of line

Shift + $

Delete single line

dd

Search

Basic search /<search pattern>

Case insensitive search /\c <search pattern>

Exit (save)

Save and quit ESC + :wq

Force quit without saving ESC + :qa!

Lorem ipsum ESC + :lorem-ipsum
